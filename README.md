# Filmotracker

Telegram bot that helps you discover and track movies and TV shows. </br>
The bot supports search in any language and notifies you about new episodes or movie premieres so you never miss a release.

### Functionality

<ul>
  <li>Search for movies and TV shows by title or keyword — works in any language</li>
  <li>
    Detailed information:
    <ul>
      <li>genre</li>
      <li>rating</li>
      <li>number of seasons and episodes</li>
      <li>next/last episode date</li>
      <li>description</li>
    </ul>
  </li>
  <li>Watch trailer</li>
  <li>Available translation languages</li>
  <li>Personal library — save titles to track future releases</li>
  <li>Release notifications — get alerts about new episode</li>
  <li>Interface available in 17 languages</li>
  <li>Smart search — automatically translates queries for better accuracy</li>
</ul>

### Tech Stack

<ul>
  <li><b>Python</b> – main programming language</li>
  <li><b>aiogram</b> – framework for building Telegram bots</li>
  <li><b>aiohttp</b> – asynchronous HTTP requests to the movie API</li>
  <li><b>Google Cloud Translate</b> – automatic translation of search queries</li>
  <li><b>PostgreSQL</b> – database for storing users and their libraries</li>
  <li><b>SQLAlchemy</b> – ORM for interacting with the database</li>
  <li><b>Stripe</b> – subscription management using webhooks</li>
  <li><b>Heroku</b> – hosting and deployment platform</li>
</ul>
